# Icon

**SelfKit** use iconify to display icons. You can fin all available icons [here](https://icon-sets.iconify.design/).

```svelte
<script>
    import Icon from '@iconify/svelte';
</script>

<Icon class="w-4 h-4 text-black" icon="lucide:external-link" />
```
